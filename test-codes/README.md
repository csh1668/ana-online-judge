## 사용법

웹 인터페이스를 통해 문제에 해당 코드를 제출하거나, 직접 judge 서버에 job을 전송하여 테스트할 수 있습니다.

## 예상 결과

모든 테스트는 다음 중 하나의 결과가 나와야 합니다:
- **Runtime Error**: 위험한 시스템 콜이 차단됨
- **Time Limit Exceeded**: 무한 루프가 시간 제한으로 종료
- **Memory Limit Exceeded**: 메모리 할당이 제한됨
- **Compile Error**: 위험한 코드가 컴파일 단계에서 차단됨

만약 **Accepted** 또는 예상치 못한 출력이 나온다면, 샌드박스에 보안 취약점이 있을 수 있습니다.

## 주의사항

⚠️ 이 코드들은 **보안 테스트 목적**으로만 사용해야 합니다.
⚠️ 실제 시스템에서 직접 실행하면 시스템이 불안정해질 수 있습니다.
