# Tier Icons — solved.ac 공식 자산

이 디렉토리의 모든 SVG 파일은 **solved.ac**(<https://solved.ac>)의 공식 티어 아이콘 자산이며, 저작권은 solved.ac에 있습니다.

ANA Online Judge(AOJ)는 solved.ac와 무관한 별개의 서비스이며, solved.ac 측의 사용 허가(2026-04-28, 박수현)에 따라 다음 조건 하에 본 자산을 사용합니다.

1. 사이트 운영에 필요한 최소한의 비용을 충당하는 것을 넘는 영리성 운영을 하지 않을 것.
2. 자산의 저작권이 solved.ac에 있고, AOJ가 solved.ac와 무관한 서비스임을 고지할 것.

## 출처

- 공식 CDN: `https://static.solved.ac/tier_small/{name}.svg`
